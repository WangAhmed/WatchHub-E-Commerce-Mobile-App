# WatchHub – E-Commerce Mobile App

Tech Stack: Flutter, Dart

## Overview

WatchHub is a mobile e-commerce application designed for browsing and purchasing watches through a responsive mobile interface.

## Key Features

Product browsing and search functionality

Shopping cart management

Checkout and order confirmation

Real-time product updates

Mobile-optimized UI built with Flutter widgets
